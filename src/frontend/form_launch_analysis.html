📋 Types d'entrée supportés

🧪 SRA (Reads bruts NCBI)

Ex: SRR28083254, SRR19561143

Pipeline complet: QC → Assemblage → Annotation → ARG

🧬 GenBank (Génome complet)

Ex: CP133916.1, NC_000913.3

Annotation → ARG (pas d'assemblage)

📦 Assembly (Assemblage NCBI)

Ex: GCA_025717695.1, GCF_000005845.2

Annotation → ARG (pas d'assemblage)

📁 FASTA local (génome assemblé)

Upload un fichier .fasta / .fna / .fa

Annotation → ARG (génome pré-assemblé)

🔬 FASTQ local (reads Illumina de votre séquenceur)

Upload R1 + R2 (paired-end) ou un seul fichier (single-end)

Pipeline complet : QC (fastp) → Assemblage (SPAdes) → Annotation → ARG
