🎯 LE PROBLÈME DE DÉPART
================================================================================

Les biologistes/chercheurs en microbiologie doivent analyser des génomes
bactériens pour détecter les gènes de résistance aux antibiotiques (ARG).
Le pipeline bash existant était très puissant mais IMPOSSIBLE à installer
pour un non-informaticien :

    Biologiste typique
          ↓
    "Installez conda"     ❌ Quoi ? Comment ?
    "Activez l'env"       ❌ C'est quoi un env ?
    "Le script plante"    ❌ Erreur ligne 2847...
          ↓
       ABANDON

🧬 MIGRATION CONDA → MAMBA (LES CHANGEMENTS TECHNIQUES)
================================================================================

AVANT (VERSION ORIGINALE)
─────────────────────────

Dans le pipeline bash:

    eval "$(conda shell.bash hook)"
    conda activate qc_arg
      fastqc fichier.fastq
    conda deactivate

    conda activate assembly_arg
      spades.py --isolate
    conda deactivate

    # ... 30+ activate/deactivate ...


APRÈS (VERSION DOCKER)
──────────────────────

Dans le pipeline bash simplifié:

    # L'env megam_arg est déjà activé par entrypoint.sh
    fastqc fichier.fastq    # Appel direct
    spades.py --isolate     # Appel direct

    # Outils isolés (conflits)
    mamba run --no-banner -n snippy_prokka prokka genome.fasta
    mamba run --no-banner -n abricate_env abricate --db card genome.fasta


MODIFICATIONS DANS LE CODE PYTHON
──────────────────────────────────

Avant (pipeline_launcher.py):

    def __init__(self, pipeline_script, work_dir, conda_init_script=None):
        # Détection conda
        conda_base = subprocess.getoutput("conda info --base")
        self.conda_init = f"{conda_base}/etc/profile.d/conda.sh"

    def build_command(...):
        cmd = f"bash {script} {sample_id}"
        # Wrapping avec conda
        full_cmd = f"source {self.conda_init} && {cmd}"
        return full_cmd


Après (version Docker):

    def __init__(self, pipeline_script, work_dir):
        # Plus de détection conda
        self.pipeline_script = Path(pipeline_script)
        self.work_dir = Path(work_dir)

    def build_command(...):
        cmd = f"bash {script} {sample_id}"
        return cmd  # Pas de wrapping, mamba déjà actif
